# Commands
1. SELECT *Column1, Column2*
2. FROM *Table*
3. WHERE *Column = Value*
4. ORDER BY *Column1, Column2* (Desc/Asc)
5. UPDATE *Table*
6. SET *Column = value*
7. INSERT INTO *Table* ***OR*** INSERT INTO *Table(Column1, Column2)* 
8. VALUES *(Data, Data)*
9. DELETE FROM *Table*
10. AND, OR, NOT
