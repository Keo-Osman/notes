# Definition
Let $a,b,c \in \mathbb{R}_{\geq_{0}}, \ r\in \mathbb{R}^{+}$, Then *Schur's Inequality* States $$\sum_{cyc}a^{r}(a^{2}+bc)\geq \sum_{cyc}a^{r+1}(b+c)$$With equality if and only if $a=b=c$ or $\{a,b,c\}=\{x,x,0\}$
The $r=1$ case is most common and is $$a^{3}+b^{3}+c^{3}+3abc \leq \sum_{sym}a^{2}b$$
